Cable Rule Fiber is light. Copper is electricity.

Cables & Media

Make cable questions easy.

Do not memorize every tiny detail first. Start with the big patterns CompTIA loves to test.

Main Memory Rules

Fiber = light = fastest, longest distance, no EMI.
Copper = electricity = common Ethernet, cheaper, affected by EMI.
STP = shielded. UTP = unshielded.

Cable Types

Fiber

  • Uses light
  • Best for long distance
  • Not affected by EMI
  • Common in backbones and data centers

Copper

  • Uses electrical signals
  • Common for Ethernet
  • Cheaper and easier to install
  • Can be affected by EMI

Single-Mode Fiber

  • Longer distance
  • Uses laser light
  • More expensive
  • Think: one path, far distance

Multimode Fiber

  • Shorter distance
  • Usually uses LED light
  • Less expensive than single-mode
  • Think: many paths, shorter range

UTP

  • Unshielded Twisted Pair
  • Very common Ethernet cable
  • Cheaper
  • Less protection from interference

STP

  • Shielded Twisted Pair
  • Better protection from EMI
  • More expensive
  • S = shield

Exam Traps

EMI Problem?

Pick fiber or shielded cable.

Longest Distance?

Pick single-mode fiber.

Normal Office Ethernet?

Think twisted pair copper and RJ45.

Air Handling Space?

Think plenum-rated cable.
